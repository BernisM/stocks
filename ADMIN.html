🔧 Jobs individuels

Action Détail Durée
Sync rapide DATA Télécharge les 5 derniers jours de prix + recalcule tous les scores. Pas d'email. Usage quotidien recommandé hors chaîne. ~5–8 min ▶ Lancer
Sync complet DATA Télécharge 1 an de prix pour tous les actifs + scores. À utiliser après ajout de nouveaux tickers ou réinitialisation. ~30–60 min ▶ Lancer
Train ML ML Réentraîne les modèles RF+XGB+LGB (4 groupes : EUROPE, US, CRYPTO, COMMO). À lancer après un sync complet ou après ajout de features. ~5 min ▶ Lancer
Sentiment news NEWS Calcule le sentiment (−100 à +100) pour tous les stocks Buy/Strong Buy via Yahoo Finance. Boost ±3/±5 pts appliqué au prochain scoring. ~3–4 min ▶ Lancer
Fondamentaux DATA Met à jour P/E, ROE, P/B, D/E, croissance CA depuis yfinance. Normalement dimanche 04h00. Relancer si données manquantes. ~6 min ▶ Lancer
Recalc Hyper-Growth DATA Recalcule les scores 🦄 sans appel yfinance. Filtre SQL pré-sélectionne ~100 stocks éligibles (rev_growth ≥ 15%, score ≥ 55, D/E ≤ 3×). ~30 sec ▶ Lancer
Refresh tickers DATA Met à jour les listes CAC40/SBF120 (Wikipedia) et NASDAQ_GROWTH (iShares IWO). Soft-delete des tickers sortis d'index. Email diff si changements. ~30 sec ▶ Lancer
