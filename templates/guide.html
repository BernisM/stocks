{# ── Scores ── #}
Scores
🎯 Score technique 0–100
{% if user.level == 'beginner' %}

Note globale de la santé technique de l'action. Plus c'est haut, plus les signaux sont positifs. ≥ 80 = fort signal d'achat.

{% elif user.level == 'intermediate' %}

Agrégation de 4 composantes : tendance (25 pts), momentum RSI/MACD (25 pts), volume/OBV (20 pts), Ichimoku (15 pts). Le modèle ML ajoute ±15 pts.

{% else %}

Tendance SMA50>SMA200 + EMA50 (25 pts) · RSI+MACD momentum (25 pts) · Volume/OBV breakout (20 pts) · Ichimoku cloud position (15 pts) · ML boost −15 à +15. Max théorique = 100.

{% endif %}
🏢 Fonda. 0–100
{% if user.level == 'beginner' %}

Note sur la santé financière de l'entreprise (bénéfices, dettes, croissance). Complète le score technique.

{% elif user.level == 'intermediate' %}

Score fondamental = P/E (30 pts) + ROE (25 pts) + P/B (20 pts) + D/E (15 pts) + Croissance CA (10 pts). Mis à jour chaque dimanche.

{% else %}

P/E <15=25pts, <25=20pts · ROE >25%=25pts, >15%=20pts · P/B <1=20pts, <2=17pts · D/E <30=15pts, <80=12pts · Growth >20%=10pts. Données yfinance .info.

{% endif %}
⚖️ Composite 0–100
{% if user.level == 'beginner' %}

Score final combinant technique et fondamental. La meilleure synthèse pour décider.

{% elif user.level == 'intermediate' %}

Composite = 65 % Score technique + 35 % Fondamental. Favorise les signaux charts tout en tenant compte des fondamentaux.

{% else %}

score_composite = round(score_final * 0.65 + fundamental_score * 0.35). Pondération calibrée empiriquement sur le backtest.

{% endif %}
{# ── Signal ── #}
Signal & Risque
🚦 Signal
{% if user.level == 'beginner' %}

Strong Buy ≥ 75 · Buy ≥ 58 · Neutral ≥ 42 · Avoid < 42. Basé sur le score technique.

{% else %}

Calculé sur score_final : Strong Buy ≥ 75 · Buy ≥ 58 · Neutral ≥ 42 · Avoid < 42. Le ML boost peut faire basculer un signal d'une catégorie à l'autre.

{% endif %}
🛑 Stop-Loss
{% if user.level == 'beginner' %}

Prix en dessous duquel vendre pour limiter les pertes. Calculé automatiquement à partir de la volatilité récente.

{% elif user.level == 'intermediate' %}

Stop = Clôture − 2 × ATR(14). L'ATR mesure l'amplitude moyenne des mouvements sur 14 jours.

{% else %}

Stop_Loss = close − 2 × ATR(14). Méthode Wilder. Déclenche une alerte email si le prix franchit le seuil en intraday (vérification toutes les 15 min).

{% endif %}
💥 Volatilité %
{% if user.level == 'beginner' %}

Mesure l'agitation du titre. Haute volatilité = risque plus élevé mais aussi opportunités plus grandes.

{% else %}

Écart-type annualisé des rendements journaliers sur 20 jours : std(returns_20j) × √252 × 100. Au-delà de 40 % = titre spéculatif.

{% endif %}
{# ── Indicateurs techniques ── #}
Indicateurs techniques
📊 RSI 0–100
{% if user.level == 'beginner' %}

Indique si l'action est trop achetée ou trop vendue. < 30 = survente (rebond possible si confirmé par le MACD), > 70 = surachat (risque de repli).

{% elif user.level == 'intermediate' %}

RSI 14 périodes (Wilder). Zone neutre 40–60. Score +12 pts si RSI 50–70. RSI <30 donne +8 pts uniquement si le MACD confirme un rebond (évite les couteaux qui tombent).

{% else %}

Relative Strength Index, fenêtre 14. RSI = 100 − 100/(1+RS). Score : RSI 50–70 = +12 pts · RSI <30 + MACD_hist >0 = +8 pts · RSI <30 seul = 0 pt · RSI >70 = −5 pts. Volume score : ratio ≥1.5 = +10 pts, ≥1.3 = +5 pts.

{% endif %}
📈 MACD
{% if user.level == 'beginner' %}

Détecte les changements de tendance. Une barre verte dans la colonne MACD signifie une dynamique haussière.

{% elif user.level == 'intermediate' %}

MACD(12,26,9). Histogramme positif = momentum haussier. Croisement MACD > Signal = signal d'achat. +15 pts si histogramme > 0.

{% else %}

EMA12 − EMA26 = ligne MACD · Signal = EMA9(MACD) · Histogramme = MACD − Signal. Score : hist > 0 = +15 pts, croisement haussier récent = bonus.

{% endif %}
🎯 Bollinger %B 0–1
{% if user.level == 'beginner' %}

Position du prix dans ses bandes de Bollinger. < 0.2 = prix bas (potentiel rebond), > 0.8 = prix haut.

{% elif user.level == 'intermediate' %}

%B = (Prix − Bande basse) / (Bande haute − Bande basse). Fenêtre 20, 2 écarts-types. 0 = touche la bande basse, 1 = touche la bande haute.

{% else %}

%B = (close − BB_low) / (BB_high − BB_low). BB(20, 2σ). Combiné avec RSI <30 = signal de rebond fort. Score : %B <0.2 = +5 pts.

{% endif %}
📏 ATR% %
{% if user.level == 'beginner' %}

Amplitude journalière moyenne du titre en %. Un ATR% de 2 % signifie que le titre bouge en moyenne de 2 % par jour.

{% else %}

ATR(14) / Clôture × 100. Mesure la volatilité en termes relatifs. Utile pour calibrer la taille de position : plus l'ATR% est élevé, plus le stop est large.

{% endif %}
🤖 ML Prob. %
{% if user.level == 'beginner' %}

Probabilité calculée par intelligence artificielle que l'action monte de +2 % dans les 10 prochains jours.

{% elif user.level == 'intermediate' %}

Random Forest entraîné sur 100k+ observations. Features : RSI, MACD, Bollinger, Volume, ATR, OBV, Ichimoku. Précision ~66.8 %.

{% else %}

RandomForestClassifier, 200 arbres. Label : close[+10] > close × 1.02. Split chronologique 80/20. Features normalisées (StandardScaler). Boost score : prob >0.65 = +15 pts, >0.55 = +8 pts.

{% endif %}
{# ── Fondamentaux ── #}
Fondamentaux
💰 P/E ratio
{% if user.level == 'beginner' %}

Prix de l'action divisé par le bénéfice annuel. Bas = moins cher. En dessous de 15, l'action est considérée bon marché.

{% elif user.level == 'intermediate' %}

Price/Earnings. Forward PE (estimations analystes) prioritaire sur trailing PE. <15 = value, 15–25 = raisonnable, >35 = croissance chère.

{% else %}

Forward PE (yfinance forwardPE) ou trailing si indisponible. Score : <10=30pts, <15=25pts, <25=20pts, <35=13pts, sinon 5pts. PE négatif → 12 pts neutres.

{% endif %}
📐 P/B ratio
{% if user.level == 'beginner' %}

Compare le prix de l'action à la valeur comptable de l'entreprise. < 1 = potentiellement sous-évalué.

{% elif user.level == 'intermediate' %}

Price/Book Value. <1 = décote sur actif net, <2 = raisonnable, >8 = prime élevée (justifiée pour les techs à forte rentabilité).

{% else %}

yfinance priceToBook. Score : <1=20pts, <2=17pts, <4=12pts, <8=6pts, sinon 3pts. Non pertinent pour les banques (utiliser P/TBV à la place).

{% endif %}
🏆 ROE% %
{% if user.level == 'beginner' %}

Rentabilité des capitaux propres. Mesure à quel point l'entreprise utilise bien l'argent de ses actionnaires. > 15 % = bien.

{% elif user.level == 'intermediate' %}

Return on Equity = Bénéfice net / Capitaux propres. >25 % = excellent (LVMH, Apple). Attention au ROE artificiellement élevé par l'endettement.

{% else %}

yfinance returnOnEquity (fraction). Score : >25%=25pts, >15%=20pts, >8%=14pts, >0=7pts, négatif=0. Croiser avec D/E pour détecter le levier.

{% endif %}
🏦 D/E ratio
{% if user.level == 'beginner' %}

Niveau d'endettement de l'entreprise. Bas = moins de dettes, donc moins de risque financier.

{% elif user.level == 'intermediate' %}

Debt/Equity. Affiché ÷100 (yfinance retourne ×100). <0.3 = peu endetté, 0.3–1.5 = normal, >3 = très endetté.

{% else %}

yfinance debtToEquity ÷ 100 pour affichage. Score : <30=15pts, <80=12pts, <150=9pts, <300=4pts, sinon 0. Seuils relatifs au secteur — les utilities et banques sont structurellement endettées.

{% endif %}
📈 Croiss.% %/an
{% if user.level == 'beginner' %}

Croissance du chiffre d'affaires sur un an. Une entreprise qui grandit vite est souvent plus prometteuse.

{% elif user.level == 'intermediate' %}

Revenue growth YoY. >20 % = forte croissance, >10 % = bonne, <0 % = déclin. À croiser avec la rentabilité : croître à perte est risqué.

{% else %}

yfinance revenueGrowth (fraction). Score : >20%=10pts, >10%=8pts, >3%=5pts, >0=3pts, négatif=0. Donnée trimestrielle glissante, peut être volatile.

{% endif %}
